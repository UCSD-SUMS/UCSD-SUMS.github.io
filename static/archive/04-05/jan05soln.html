Solution by Mike Skirvin

Suppose X != f(X) and let y be an element of X - f(X). X compact and f(X) continuous => f(X) compact. Furthermore, X metric space => X is Hausdorff, so f(X) is closed => X - f(X) is open. Therefore we can find e > 0 s.t. the ball of radius epsilon around y is disjoint from f(X).

Define x_1 = y and x_n = f(x_(n-1)). We will show that d(x_n, x_m) >= e for all n != m by induction on n. If n = 1, x_1 = y, and x_m is in f(X) for m != 1 => d(y, x_m) > e.

Assume d(x_(n-1), x_m) > e for n-1 != m. Then e > d(x_(n-1), x_m) =(by isometry) d(f(x_(n-1), x_m) = d(x_n, x_m).

Then the sequence {x_n} cannot have a convergent subsequence, contradicting X being compact. So X = f(X) and f is surjective.

For injectivity, if f(x) = f(y), then 0 = d(f(x), f(y)) = d(x, y) => x = y.
