Solution by Mike Skirvin

Part (a):
If M is finite and pq = e, consider the map M -> M given by x -> qx. This map is injective since qx = qy => x = y by left multiplication of p. Since M is finite the map is onto. So there is some v in M s.t. qv = e. Now left multiply by p, hence getting v = p. So qp = e.

 Part (b):
Consider the set of all functions from N = {1, 2, 3, ...} naturals to itself. This set is monoid under composition of map. Let g(x) = 2x, and f(x) = x/2 if x is even, and x if x is odd. One easily verifies that fg = id but gf is clearly not (it's not even onto!).
