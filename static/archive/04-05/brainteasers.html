April's Brainteaser
Let p_1, p_2, ..., p_n be a sequence of distinct primes. Show that sqrt(p_1), ..., sqrt(p_n) are linearly independent over the rationals, i.e. show that whenever x_1*sqrt(p_1) + ... + x_n*sqrt(p_n) = 0 for x_1, ..., x_n rational numbers, x_1 = x_2 = ... = x_n = 0.

For example for n = 2, p_1 = 2 and p_2 = 3, it is easy to see that sqrt(2) and sqrt(3) are linearly independent over the ratioanls, for if a*sqrt(2) + b*sqrt(3) = 0 with a,b ratioals and a != 0, say, then b/a = -sqrt(2/3), contradiction as left side is rational whereas right side is irrational.
