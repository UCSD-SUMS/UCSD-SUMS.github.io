Solution by Michael Viscardi

Define b_n = sqrt(n)*a_n and c_n = 1 / sqrt(n) for all natural numbers n. Note that:
i) The partial sums of SUM b_n form a bounded sequence (since SUM b_n converges).
ii) c_0 >= c_1 >= c_2 >= ...
iii) c_n --> 0 as n --> infinity.
So by Abel's Convergence Theorem, SUM b_n*c_n = SUM a_n converges, as desired.
